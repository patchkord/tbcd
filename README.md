# Tbcd

A NIF library for decoding and encoding of Telephony binary-coded decimal (TBCD)

## Installation

The package can be installed by adding `tbcd` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:tbcd, "~> 0.1"}]
end
```

## Build

```
    $ rebar3 compile
```

or

```
    $ mix compile
```
